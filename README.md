This program is very simple. It is useful for someone who wants to keep track of when they take their medications. But it could be used for anything that you need to track the date and time.

The first time the program runs, it asks what day of medication the user is on. This is for if someone started using it after they've already started their medication. Then it creates a text file on the user's desktop and logs the date and time and whatever day you entered.

Any subsequent time the program runs, it should simply increment the day and then log the date and time that the script ran in the text file.

The intended point of the script is to just run it right after taking your medication to log the day, date, and time you took it. It is very quick and easy
